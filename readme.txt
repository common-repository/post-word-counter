== Description ==
Post Word Counter is a WordPress plugin to show the word counter and time counter in single post.


== Installation ==

= Minimum Requirements =

* WordPress 6.1 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Post Word Counter


== Frequently Asked Questions ==

**How do I install Post Word Counter?**

From your WordPress dashboard -> Go to Plugins -> Click on **Add new**. In the Search field, enter **Post Word Counter** and install **Post Word Counter**. After installation, click **Activate**.

**Does Post Word Counter work with all the themes?**

Post Word Counter works all the themes that respect the coding standards of WordPress set by its Codex. However, if you get any bug, please open support ticket to let me know. I will be happy to support.

**Is Post Word Counter compatible with Gutenberg?**

Post Word Counter and Gutenberg work seamlessly together.

**Does it work with other WordPress plugins? **

It works with almost all the plugins. If you experience an incompatibility issue, please report it to us and to the plugin that conflicts with Post Word Counter.

**Do I need to know how to code?**

No! Post Word Counter provides very simple settings options. We are working for more.

**Will Post Word Counter slow down my website?**

No.

**Is my site secure with Post Word Counter?**

The security of your website is extremely important to me and we take proactive measures to assure that your websites are secure.

**Is Post Word Counter compatible with Page and Custom Post Types?**

Not yet. We are working on it.
